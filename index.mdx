LLM Council is a local web application that replaces single-model queries with a structured deliberation process. Submit a question, and the council—GPT, Gemini, Claude, Grok, or any OpenRouter-supported model—answers independently, ranks each other's work under anonymous labels, and lets a designated Chairman synthesize the collective wisdom into a single final response.

## The 3-Stage Process

<Steps>
  <Step title="Stage 1 — First Opinions">
    Your query is sent to every council model in parallel via OpenRouter. Each model answers independently, and all responses are displayed side-by-side in a tab view so you can inspect them.
  </Step>
  <Step title="Stage 2 — Anonymous Peer Review">
    Each council model receives all Stage 1 responses with model identities stripped (Response A, Response B, …). It evaluates every response and provides a `FINAL RANKING:` in a structured format. The app computes aggregate rank scores across all evaluators.
  </Step>
  <Step title="Stage 3 — Chairman Synthesis">
    The designated Chairman model receives all individual responses plus all peer rankings and distills everything into one comprehensive final answer—the council's collective verdict.
  </Step>
</Steps>

## Why LLM Council?

<CardGroup cols={2}>
  <Card title="Reduce single-model bias" icon="scale-balanced">
    No single model's blind spots dominate. Peer review surfaces weaknesses before the final answer is written.
  </Card>
  <Card title="Full transparency" icon="eye">
    Every raw response and every raw ranking evaluation is inspectable. Nothing is hidden behind an abstraction.
  </Card>
  <Card title="OpenRouter-powered" icon="network-wired">
    Access GPT, Gemini, Claude, Grok, and hundreds of other models through a single API key—no juggling multiple provider accounts.
  </Card>
  <Card title="Local & private" icon="lock">
    The app runs entirely on your machine. Conversations are stored as JSON files locally—no cloud sync, no telemetry.
  </Card>
</CardGroup>
